# Virtual Bingo Frontend Demo

This is the Next.js demo app built for manager review.

It is useful as a UI reference, but it is not the final production architecture. The production plan is documented in:

- `../../docs/proposals/TECH_STACK_PROPOSAL_UPDATED.md`
- `../../docs/architecture/AUTONOMOUS_BACKEND_ARCHITECTURE.md`
- `../../docs/architecture/GO_BACKEND_PLAN.md`

## Run Locally

```bash
npm install
npm run dev
```

## Notes

- The app uses mock data only.
- Authentication, recurring games, Microsoft Graph delivery, Go backend APIs, AI services, rewards, and realtime state are future production work.
- The previous AI Studio/Gemini package/env scaffold has been removed, but the app still uses mock data only.
